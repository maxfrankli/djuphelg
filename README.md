# Djuphelg

Detta repository innehåller webbplatsen och projektdokumentationen för **Djuphelg – Gullmarn**.

Djuphelg är en deltagardriven fridykningshelg där initiativtagaren tillhandahåller ram, säkerhet och grundläggande logistik, medan deltagarna tillsammans skapar innehållet.

## Webbplats

Webbplatsen är en statisk sida byggd i HTML och CSS och publiceras via GitHub Pages.

## Projektdokumentation

Dokumentationen finns i mappen [`docs`](docs/).

Börja här:

- [`docs/ai-context.md`](docs/ai-context.md) – sammanfattning för en ny AI eller projektmedlem
- [`docs/vision.md`](docs/vision.md) – vision, filosofi och principer
- [`docs/event.md`](docs/event.md) – upplägg för pilothelgen
- [`docs/website.md`](docs/website.md) – innehåll och designprinciper för webbplatsen
- [`docs/technical.md`](docs/technical.md) – teknisk lösning och DNS
- [`docs/decisions.md`](docs/decisions.md) – fattade beslut
- [`docs/todo.md`](docs/todo.md) – aktuella nästa steg

## Projektstatus

Pilothelgen planeras till **14–16 augusti 2026** med bas på **Fossens Camping vid Gullmarn**.
